# ‚ö° **1) `throw` ‚Üí You *throw* an exception manually**

### ‚úî Used **inside a method/block**

### ‚úî Creates and throws an exception object

### ‚úî Only **one** exception at a time

Example:


üëâ *‚Äúthrow = I throw the exception right now.‚Äù*

# ‚ö° **2) `throws` ‚Üí You *announce* that a method may throw exception(s)**

### ‚úî Used **in method declaration**

### ‚úî Tells the compiler that this method might throw one or more checked exceptions

### ‚úî Does NOT throw anything by itself

Example:

```java

    // method may throw IOException
}
```

**Think:**
üëâ *‚Äúthrows = I am warning you that this method might throw exceptions.‚Äù*

---

# ‚ö° **3) `Throwable` ‚Üí Parent class of ALL errors & exceptions**

### ‚úî Superclass of:

* `Exception`
* `Error`

### ‚úî Can be used to catch **anything** (not recommended normally)

Example:

```java

    System.out.println("Caught anything!");
}
```

**Think:**
üëâ *‚ÄúThrowable = Daddy of both Errors and Exceptions.‚Äù*

# üéØ **SUPER EASY TRICK TO REMEMBER**

### **throw ‚Üí action**

You *throw* the ball (exception).

### **throws ‚Üí warning**

You *warn* that you **might** throw the ball.

### **Throwable ‚Üí family**

The entire **family** of things that can be thrown.
